## Jetbrains系IDE 新窗口打开链接

Jetbrains系列IDE默认开启了 `Enable preview tab`， 会导致创建一个临时的窗口用于展示代码链接内容。然而，当想查看下一个代码链接时，内容也会在该窗口中显示而不是打开新窗口。

需要在`setting->Editor->Tab Order`里取消勾选 `Enable preview tab`
